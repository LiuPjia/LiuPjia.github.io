读文件

读写文件使用with语句，避免处理异常

方法 含义
f.read() 读取文件的全部内容
f.read(size) 最多读取size个字节
f.readline() 每次读取一行内容
f.readlines() 一次性读取所有内容，并按行返回list

写文件

写文件和读文件的方法类似，只是参数不同

  • w参数是覆盖，每次都会覆盖掉文件
  • 使用a，append，追加文件
