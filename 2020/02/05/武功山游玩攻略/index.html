武功山两日游玩攻略

发表于 2020-02-05 分类于 旅游计划

武功山简介

武功山位于江西省萍乡市，是一个比较小众的5A级景点。武功山的主峰金顶海拔1900米，有高山瀑布，云海日出，类似于草原的地貌，还有保存完好的原始森林。我们如果去的话应该是将近三月了，那时天气比较暖和适宜。：

行程

第一天路线

第一天路线就是从景区大门到金顶，不赶时间。上山的话有两条途径，一条是通过缆车，一条是步行，完全步行的话大约需要三、四个小时（你个小懒猫还是算了吧，哈哈）。

  • 第一天路线A(坐两次缆车)

    游客中心乘交通车-中庵缆车下站-坐索道到紫极宫(上站)-许愿桥福星谷栈道-二级索道-金顶。

  • 第一天路线B (坐一次缆车)

    游客中心乘交通车-中庵缆车下站-坐索道到紫极宫(上站)-许愿石-情人花廊武功神拳金顶。

第二天路线

第二天我们就可以早起起床看日出，8点左右可以下山。可以游玩这些景点：

金顶看日出 - 万宝柜 - 吊马桩 - 玻璃栈道 - 返回吊马桩 - 好汉坡 - 紫极宫( 中庵索道上站) - 坐索道到石鼓寺交通车到游客中心。

注意事项

1. 住宿

我们这次是要在山顶上过夜，上面有客栈，也可以选择租借帐篷，嘿嘿，可以选择帐篷野营呀。

2. 天气

出行看好天气预报，选个晴朗的天气。如果选择露营的话，昼夜温差肯定很大，我们要注重保暖措施，准备好热水，暖宝宝，热水袋。

3. 出行

如果是从淮北出发，做火车需要13个小时，下午四点到早上两点，车票卧铺230，但是这样有一个缺点，因为我们从家里出发是有行李的，所以我建议是先从淮北-武汉，把行李放好，轻装上阵，再从武汉到萍乡，从武昌坐车凌晨2点，大约5个小时。但是就是比较麻烦，如何解决行李的问题呢？(有待商榷)从车站下车后，可以坐1路、2路a、9路，12路等公交，到达公交站点（城南客运站）下车，之后坐客车到武功山脚下。

4. 准备

除了上面说的防寒措施外，还要准备很多食物，因为上山后上面的食物贵且不好，可以多准备点牛肉干，坚果等高热量的食物。你可以准备一顶帽子，上面的风肯定很大，穿着方便一点，因为是要去爬山滴。

5. 返回

玩了两天后，我们要做下午三点的车回去，所以我们根据上山的时间推测出下山到车站的时间，很重要！！不要晚点了，这两天你肯定很累，订间民宿，饱餐一顿，然后我再给你按按腰，捏捏小脚丫。嘻嘻。

开销

车票700，住宿三天600，门票+缆车出行250，吃饭400，所以大约需要2000块钱。
